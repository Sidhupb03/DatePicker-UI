# DatePicker-UI
Datepicker Calendar UI

A clean and simple HTML + CSS datepicker calendar UI, designed without JavaScript.
Built using CSS Grid, this component displays dates in a familiar 7-column calendar format and includes borders around each day cell for a professional, clean layout.
